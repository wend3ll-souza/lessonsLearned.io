Sobre conhecimento

Um conhecimento nunca é concreto, sempre se tem algo mais a aprender. Os exemplos contidos nessa página são resumos do que experienciei esse mês. Não tome-os como conclusivos, pois cada tópico é bem mais profundo do que o conteúdo dessas pequenas amostras. Aqui você irá encontrar um incentivo para utilizar materiais como este como rota de estudos. Por fim, deixarei o link para documentações de cada tema em seus títulos.

Divirtam-se.

GNU-Linux/ Bash

O popular Linux é uma tecnologia Unix Like. Junto ao projeto GNU deu origem a dezenas de Distribuições. Usamos nele o Shell,que permite o contado direto com o nucleo do sistema. Tornando possível de apartir de um Shell Script fazer alterações no sistema operacional.

Git

Uma ferramenta essencial para o trabalho do desenvolvedor é o Git. Este programa é usado diariamente para versionamento de códigos. Esse artifício traz mais segurança e controle sobre os projetos. Podendo se, assim, checar alterações feitas nas versões anteriores.

GitHub

Assim como o Git, o GitHub é uma ferramenta que também auxilia no controle de versionamento de nossos codigos. Nele se pode criar um repósitorio remoto que será vinculado ao repositório local,podendo então fazer atualizações em ambos. Além do mais, o GitHub é uma plataforma que ajuda, e muito, em trabalhos em grupo e na troca de conhecimento. Nele é possivel deixar seu repósitorio público, fazendo com que outras pessoas possam baixar-lo e contribuir com o projeto. Tudo isso com sua autorização, é claro. O GitHub é acima de tudo uma ferramenta de troca de conhecimento!

HTML

O HTML é a Markdown que constroi a estrutura nos Desenvolvimentos Web. Atua como um documento que sera traduzido pelo browser, tento como retorno o conteudo contido nele. Nele é possivel colocar inumeros conteudos, como textos, links, imagens, entre outras midias. É possivel também colocar links para conteúdos internos e externos, assim podendo criar rotas para outros documentos, como o de estilização , o CSS.

CSS

O CSS posiciona e estiliza os elementos contidos em uma página HTML. Dessa maneira trazendo novas cores, fontes, e até animações. Tonando o trabalho final mais agradavavel aos olhos.

Internet

A internet é formada por uma rede de computadores espalhados pelo mundo. É possivel, com um provedor de internet e um cliente, fazer requisições aos servidores e receber o conteúdo solicitado. Essas informações viajem via protocolo TCP/IP, esse gerência a transferência de informações na web.

VS code

O Vs Code é uma ótima ferramenta quendo se trata de editar códigos. Isso poderia ser feito em outros editores de texto. Porém o Code traz consigo uma serie de pluggins que ajudam a desenvolver o trabalho com mais eficiência. A experiência de usar esse ferramenta no dia a dia tem resultados percepitiveis de produtividade.

Soft Skills

Soft Skills é uma das coisas que se aprende na prática. Na Trybe podemos viver cotidianamente em um ambiente que estimula essa habilidade. Boas práticas, educação, e ajudar o próximo, sempre com impatia, são alicerces que matém uma equipe em pé. Tendo como resutado melhor desempenho nos projetos planejatos.

Semantic HTML

Assim como toda linguagem, o HTML também tem seus sotaques. Pode se ter o mesmo resultado com variados códigos, porém de nível de compreensão diferente. Um código limpo é mais maleavel para alterações e compreensiveis para trabalhos em grupo.

Chrome DevTools

O Chrome DevTools é uma ferramenta essencial e de fácil acesso no dia a dia de um Desenvolvedor Web. Com ele é possível testar previamente estilos de configurações e fixa-las no arquivo original.
